# Basic Literals

Basic literals represent fundamental data values including numbers, strings, and boolean constants. The `null` literal represents the absence of a value.

```
Literal          = NumberLiteral
                 | StringLiteral
                 | "true"
                 | "false"
                 | "null" ;
```

**Examples:**
```jyro
42
3.14159
"Hello, World!"
true
false
null
```
